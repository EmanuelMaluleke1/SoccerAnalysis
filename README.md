# Soccer Analyst Platform

A modern football analytics platform built with .NET 10 and React.

## Vision

The Soccer Analyst Platform aims to become a comprehensive football intelligence platform that provides historical football data, analytics, machine learning predictions, explainable AI, and interactive insights.

Version 1 focuses on building a reliable football data platform by ingesting, storing, and visualising football data.
